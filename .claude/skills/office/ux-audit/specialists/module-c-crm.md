## 双视角评审基准

**PM视角：** 首屏决策信息、高频操作路径深度、信息层级、模块优先级
**深度用户视角（B2B高级销售经理）：** P0字段首屏可见、
3秒内找到核心状态、四个快捷操作可触达

---

## C1 — CRM 核心字段首屏可见性

**适用：客户/商机/联系人详情页**

以下字段必须在 800px 视口高度内不需要滚动即可见。
**任何一项不可见 → P0 问题。**

- [ ] 客户名称 / 商机名称
- [ ] 当前阶段 / 成交状态
- [ ] 负责人
- [ ] 最后跟进时间
- [ ] 商机金额（商机页）/ 客户等级（客户页）
- [ ] 快捷操作按钮区（新建跟进记录、拨打电话、发送邮件）

---

## C2 — CRM 高频操作路径深度

以下操作从当前页面出发，点击次数：
- ≤ 2 次：合格
- > 2 次：**P1 问题**
- 入口完全不存在：**P0 问题**

- [ ] 新建销售跟进记录
- [ ] 推进商机阶段
- [ ] 新建联系人
- [ ] 发起审批
- [ ] 查看关联合同

---

## C3 — 数据可信度（CRM 专属）

- [ ] 每个数据字段有最后更新时间或数据新鲜度标识
- [ ] 金额字段单位明确（元/万元不混用）
- [ ] 日期格式全页统一
- [ ] 空字段有占位文本（「暂无」「未填写」），不显示空白或 null
- [ ] 异常数据有视觉标记（超期未跟进、金额异常等）
- [ ] 数字超过万位有千分位分隔符

---

## C4 — CRM 列表页专项

**适用：列表/看板页（如当前页面不是列表页，跳过此节并注明）**

- [ ] 列表支持按关键字段排序（金额、时间、阶段）
- [ ] 筛选条件保持状态（离开再回来，筛选不重置）
- [ ] 批量操作入口在选中后才出现（不占常驻空间）
- [ ] 空列表状态有明确提示和引导操作
